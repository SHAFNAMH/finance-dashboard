# 💰 Finance Dashboard

A modern and responsive Finance Dashboard to track income, expenses, and balance with interactive charts.

## 🚀 Features
- 📊 Visual charts using Chart.js  
- 💵 Income & Expense tracking  
- 🔍 Search transactions  
- 👤 Role-based view (Admin / Viewer)  
- 📱 Mobile-friendly design  

## 🛠 Tech Stack
- HTML  
- CSS  
- JavaScript  
- Chart.js
